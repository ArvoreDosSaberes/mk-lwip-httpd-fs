Quando solicitado criar slides em arquivos md, crie uma pasta para o tema em ./docs/slides/<titulo da apresentação>, nesta pasta crie:
  index.md      # (opcional) Front matter para título global
  01-intro.md   # Slide 1
  02-tema.md    # Slide 2
  nn-tema.md   # slides nn
...
  nn-final.md   # Slide 3

Use DIV  para rederizar tabelas transparentes de forma a organizar o conteúdo do slide, construindo frames em diversos formatos, escolha os formatos conforme os conteúdos.
